# TODO-CLI PHP

## Описание

Попробовать написать свой PHP аналог Console из framvork Laravel. И в дальнейшем на его основе построить приложение, позволяющее пести свой todo список. До этого не особо много работал с консольными приложениями, поэтому буду придумывать что-то на ходу.

## Структура проекта

Основной код данного проекта находится в папке ```app/```.

В данном проекте планирую использовать:
- Docker
- PHP 8.5
- MySQL/MariaDB
