# TODO-CLI PHP

## Описание

Попробовать написать свой PHP аналог Console из framvork Laravel. И в дальнейшем на его основе построить приложение, позволяющее пести свой todo список. До этого не особо много работал с консольными приложениями, поэтому буду придумывать что-то на ходу.

## Структура проекта

Основной код данного проекта находится в папке ```app/```.

В данном проекте планирую использовать:
- Docker
- PHP 8.5
- MySQL/MariaDB

## Разработка с автообновлением (watch)

Чтобы изменения кода на хосте сразу попадали в контейнер без пересборки, запускайте:

```bash
docker compose watch
```

или

```bash
docker compose up --watch
```

Обычный `docker compose up -d` **не** включает синхронизацию — в этом режиме в контейнере остаётся только то, что было скопировано при сборке образа.
